## Introduction
Boids is a simulation of a flock of birds performed on unity using job systems.
This project is open source.

## Packages used
 - Unity Version 2021.3.21f1
 - Unity Collections
 - Unity Burst
 - Unity Text Mesh Pro

## Design
The design was borrowed from Ben Eater: https://eater.net/boids
